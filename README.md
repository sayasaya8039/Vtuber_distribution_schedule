# VTuber Schedule Calendar

複数VTuberの配信スケジュールをGoogle Calendarに自動統合するChrome拡張機能 🎭📅

## 機能

### 配信スケジュール取得
- **ホロライブ公式**: ホロジュール（schedule.hololive.tv）から自動スクレイピング
- **にじさんじ公式**: にじさんじ公式サイト（nijisanji.jp/streams）から自動取得
- **Holodex API**: その他のVTuberはHolodex APIで取得
- **組織バッジ表示**: ホロライブ/にじさんじ/インディーを色分け表示

### Google Calendar連携
- **ワンクリック追加**: 📅ボタンで配信をカレンダーに追加
- **自動追加機能**: 新しい配信を検出したら自動でカレンダー登録
- **.icsエクスポート**: Outlook/Apple Calendar対応

### ゲリラ配信検知
- **定期チェック**: 設定間隔（デフォルト1時間）で自動チェック
- **プッシュ通知**: 新しい配信が追加されたら通知
- **自動カレンダー登録**: 検出と同時にカレンダーに追加（オプション）

### UI機能
- **Side Panel**: サイドパネルでスケジュール一覧を表示
- **フィルター機能**: 日付・組織別で絞り込み
- **VTuber登録**: スケジュールから直接VTuberを登録可能
- **組織別カラー**: hololive(水色)、にじさんじ(赤)で色分け

## スクリーンショット

### スケジュール一覧
- 配信時間、VTuber名、タイトルを一覧表示
- 組織バッジ（ホロライブ/にじさんじ）
- 登録済みVTuberには ✓ マーク
- 未登録VTuberは + ボタンで簡単登録

### 設定画面
- Google カレンダー連携
- データソース選択（ホロライブ/にじさんじ）
- 通知・自動同期設定

## セットアップ

### 1. 拡張機能のインストール
1. `chrome://extensions` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」
4. `VtuberScheduleCalendar` フォルダを選択

### 2. Google Calendar連携（任意）
1. 設定画面で「Googleカレンダーに接続」をクリック
2. Googleアカウントでログイン
3. カレンダーへのアクセスを許可

### 3. Holodex APIキー（任意）
ホロライブ・にじさんじ以外のVTuberを追加する場合：
1. [Holodex](https://holodex.net/login) でアカウント作成
2. 設定からAPIキーを取得
3. 拡張機能の設定画面でAPIキーを入力

## 使い方

### 基本操作
1. Chromeの拡張機能アイコンをクリック → サイドパネルが開く
2. 「スケジュール」タブで配信予定を確認
3. 📅ボタンでカレンダーに追加 / ▶ボタンでYouTubeを開く

### VTuberの登録
- スケジュール一覧で「+」ボタンをクリック → 自動登録
- または「VTuber」タブでHolodex検索して追加

### フィルター
- 日付: 今日 / 明日 / 今週 / 全期間
- 組織: 全グループ / hololive / にじさんじ / インディー

## 設定項目

| 設定 | 説明 | デフォルト |
|------|------|----------|
| ホロジュールから取得 | ホロライブ公式スケジュールを取得 | ON |
| 全ホロライブ配信を表示 | 登録関係なく全員表示 | OFF |
| にじさんじ公式から取得 | にじさんじ公式スケジュールを取得 | ON |
| 全にじさんじ配信を表示 | 登録関係なく全員表示 | OFF |
| 新しい配信を自動でカレンダーに追加 | ゲリラ配信を自動登録 | OFF |
| 更新間隔 | バックグラウンドチェック間隔 | 60分 |

## 技術スタック

- **フレームワーク**: React 19 + TypeScript
- **ビルド**: Vite 6
- **状態管理**: Zustand
- **Chrome拡張**: Manifest V3
- **API**: Holodex API, Google Calendar API
- **スクレイピング**: ホロジュール, にじさんじ公式

## ディレクトリ構成

```
Vtuber_distribution_schedule/
├── src/
│   ├── background/       # Service Worker（定期チェック）
│   ├── components/       # Reactコンポーネント
│   ├── lib/             # API・スクレイパー・ストア
│   ├── popup/           # ポップアップUI

│   └── types/           # 型定義
├── public/              # 静的ファイル・manifest.json
└── VtuberScheduleCalendar/  # ビルド出力
```

npm run dev
